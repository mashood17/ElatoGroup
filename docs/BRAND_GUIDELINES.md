# Brand Philosophy

ELATŌ is not simply a café, an event hall, or a serviced apartment.

It is a premium hospitality destination where people celebrate life's most meaningful moments.

Every design decision should communicate elegance, warmth, refinement, and confidence.

The website should feel timeless rather than trendy.

Luxury should come from restraint, quality, spacing, typography, photography, and motion—not decoration.

---

Never introduce random colors.

Avoid saturated colors.

Avoid pure black.

Avoid pure white.

Avoid colorful gradients.

# Spacing Philosophy

Whitespace is a luxury element.

Never compress layouts.

Maintain generous breathing room.

Use consistent spacing throughout.

Content should never feel crowded.

---

# Grid System

Desktop

12-column layout

Tablet

8-column layout

Mobile

4-column layout

Maintain alignment across all sections.

---

# Border Radius

Use soft premium corners.

Avoid:

- Sharp edges
- Excessive rounding

Maintain consistency throughout the website.

---

# Shadows

Use soft shadows.

Low opacity.

Large blur radius.

Never use harsh black shadows.

Shadows should communicate depth—not decoration.

---

# Motion Philosophy

Motion should feel:

- Intentional
- Calm
- Premium
- Cinematic
- Organic

Never:

- Bounce
- Shake
- Spin
- Flash
- Overshoot excessively

Every animation should support storytelling.

---

# Hero Principles

Hero is the emotional entry point.

The hierarchy must always be:

1. ELATŌ Logo
2. Main Headline
3. Supporting Copy
4. Primary CTA

No unnecessary elements.

No decorative cards.

No visual clutter.

The logo is the primary focus.

---

# Animation Principles

Use Framer Motion.

Animations should be subtle.

Preferred duration:

300–900ms

Hero entrance:

800–1400ms

Page transitions:

400–700ms

Section reveals:

400–700ms

Hover:

150–250ms

Scroll animations should feel smooth and continuous.

Respect prefers-reduced-motion.

---

# Components

Every component should feel:

- Premium
- Consistent
- Reusable
- Minimal

Avoid one-off component designs.

# Cards

# Navigation

Navigation should feel calm.

Balanced spacing.

Large clickable areas.

Smooth hover transitions.

Never overcrowded.

---

# Icons

Use Lucide icons.

Avoid filled icons unless necessary.

---

# Photography

Images should be:

Professional

High resolution

Natural lighting

Warm tones

Luxury hospitality

Elegant interiors

Premium desserts

Authentic experiences

Stock-looking photography

Low resolution

Heavy filters

---

# Responsive Design

Desktop first.

Perfect tablet.

Perfect mobile.

No clipping.

No overflow.

No horizontal scrolling.

Animations must also work on mobile.

Maintain touch-friendly spacing.

---

# Accessibility

Maintain sufficient color contrast.

Readable typography.

Keyboard accessibility.

Screen reader compatibility.

Respect reduced motion.

---

# Performance

Use optimized images.

Lazy load below-the-fold media.

Avoid layout shifts.

Maintain smooth 60fps animations.

Keep interactions responsive.

---

# Brand Consistency Checklist

Before shipping any UI:

- Does it feel premium?
- Does it feel timeless?
- Is the logo the visual priority where appropriate?
- Are colors consistent?
- Is spacing generous?
- Is typography elegant?
- Are animations subtle?
- Does it feel calm?
- Is it responsive?
- Does it strengthen the ELATŌ identity?

If any answer is "No", refine the design before considering the feature complete.
